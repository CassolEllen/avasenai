# 🎓 SENAI AVA - Plataforma Acadêmica

Projeto desenvolvido com foco em recriar e modernizar a experiência do AVA (Ambiente Virtual de Aprendizagem) do SENAI, trazendo uma interface mais intuitiva, responsiva e centrada na experiência do usuário.

---

## 🚀 Sobre o Projeto

O **SENAI AVA** é uma aplicação web que simula um ambiente acadêmico completo, permitindo que alunos:

- 📚 Acompanhem disciplinas e progresso
- 📝 Visualizem e enviem atividades
- 💬 Interajam com professores via chat
- 📄 Baixem documentos acadêmicos
- 🔔 Recebam notificações importantes
- 👤 Gerenciem seu perfil

O objetivo principal é melhorar a usabilidade, acessibilidade e fluidez da plataforma em comparação com sistemas tradicionais.

---

## 🧠 Funcionalidades

### 📊 Dashboard
- Visão geral do desempenho
- Indicadores de progresso
- Atividades pendentes
- Próximas aulas

### 📘 Disciplinas
- Listagem de matérias
- Progresso individual por disciplina
- Detalhes como ementa e carga horária

### 👨‍🏫 Professores
- Perfil com informações de contato
- Disciplinas ministradas
- Acesso direto para comunicação

### 💬 Chat
- Conversas entre aluno e professor
- Histórico de mensagens
- Possibilidade de exclusão de chat

### 📝 Atividades
- Listagem de trabalhos
- Visualização de detalhes
- Upload de arquivos (PDF, DOC, etc.)
- Submissão simplificada

### 📄 Relatórios
- Download de documentos como:
  - Boletim Escolar
  - Histórico de Aulas
  - Declaração de Matrícula

### ⚙️ Configurações
- Preferências do usuário
- Ajustes de interface
- Controle de notificações
